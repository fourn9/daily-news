Thinking Machines Lab、初のオープンウェイトモデル「Inkling」公開 ─ MoE 975B総パラメータ(41B活性)、45兆トークン学習、Nvidia Nemotron 3 Ultraの1/3トークンで同等コード性能

参照元: TechCrunch (Connie Loizos) / Thinking Machines Lab

元OpenAI CTOのミラ・ムラティ氏が創業したThinking Machines Labが7月15日、初の自社AIモデル「Inkling」を公開、しかもオープンウェイトとしてリリースした。OpenAI/Anthropic/Googleの旗艦モデルと異なり、外部開発者・企業がダウンロードして直接改変できる。技術仕様はMoE（Mixture-of-Experts）方式で総パラメータ975B・活性化41B、テキスト・画像・音声・動画の45兆トークンで学習、4モダリティ横断で推論するが出力はテキスト（コード・スタイル化・構造化データ）に限定。1年半の水面下でのAIインフラ構築後の初の公表成果で、5月の"interaction model"研究プレビュー（傾聴・発話・割り込みができるAI）に続く展開。1つのベンチマークでNvidia Nemotron 3 Ultra（最新世代オープンウェイト）の1/3トークンで同等コード性能を達成、「不確実性を明示的にフラグする校正済み回答」「thinking effortダイヤル調整」が特徴。ただし公式資料で"現在利用可能な最強モデルではない"と明言、"組織が自ら適応させるカスタマイズ可能性"に賭ける戦略で、"one-size-fits-all"の閉じたフロンティアモデルへのアンチテーゼを打ち出した。

Microsoft、月例Patch Tuesdayで史上最多570件のセキュリティ脆弱性を修正 ─ AI活用で"数十年埋もれていた"バグを発掘、SharePoint 0-dayは米CISA既に悪用警告

参照元: TechCrunch (Zack Whittaker) / Krebs on Security / Microsoft / 米CISA

Microsoftが7月14日（Patch Tuesday）に月例セキュリティ更新で史上最多となる570件の脆弱性修正を配布した。うち少なくとも2件はゼロデイ（発覚前に既に悪用）で、Windows Server脆弱性は限定ユーザーからシステム管理者への権限昇格を許し、SharePointファイル共有サーバー脆弱性は米CISA（サイバーセキュリティ・インフラセキュリティ庁）が既に"組織侵害に悪用されている"と警告済み。同社のWindows担当Pavan Davuluri氏は事前ブログで"AIが防御側のバグ発見を助けるほど、顧客は各セキュリティ更新の件数増加を目にすることになる"と説明。Windowsコードの一部は数十年前のもので、AIモデルの高度化・セキュリティ特化により長年休眠していた脆弱性が次々発掘される段階に入った。前日のIBM Q2ミスがサイバーセキュリティ懸念を"CapEx再優先"の理由に挙げていたことと呼応し、"AIが攻撃側にも防御側にも構造変化をもたらす"論点の物証。企業IT部門は570件を短期間でパッチ適用する運用負荷に直面する一方、SharePoint 0-day悪用は日本国内でも警戒対象となる。

本日最大Stripe + Advent International、PayPal買収$534億で共同提案 ─ 銀行融資約$500億コミット、両社均等出資構造、PayPal株は+18%急騰、決済業界史上最大級M&A

参照元: TechCrunch (Lauren Forristal) / Reuters / ChosunBiz

米決済スタートアップStripeとプライベートエクイティ大手Advent Internationalが、共同でPayPalを約$534億（銀行融資約$500億コミット済み）で買収する提案を今月既に提出したとReuters報道。両社が均等出資でPayPalを共同所有する構造。過去には2月にStripeが単独でPayPal買収の初期協議を行ったと報じられたが正式提案には至らず、今回が本格化した形。実現すればデジタル決済の2大巨頭の統合となり、PayPal（アクティブアカウント約4.4億・2025年決済取扱高$1.8兆）とStripe（同期間$1.9兆処理、時価総額$1,590億）が一体化する。PayPalは3月にEnrique Lores氏が業績警告後にCEO就任、2〜3年で$15億のコスト削減、人員約20%削減を進行中の再編局面。米市場でPayPal株は買収報道で+18.12%急騰、Stripeの高評価と併せて"閉じた決済プラットフォーム統合による超巨大化"の号砲となる可能性。前日IBM Q2ショックのソフト株売り圧力とは対照的に、フィンテック業界は"AI CapExを補完する決済インフラ集約"への評価で沸騰。

Anthropic × Blackstone JV「Ode」、$15億でAI実装事業を正式ローンチ ─ Fractional AI買収を基盤、Hellman&Friedman・Goldman Sachs参画、"数兆ドル企業"の野心、OpenAI "The Deployment Company"と激突

参照元: TechCrunch (Rebecca Bellan) / Anthropic / Blackstone / Fractional AI

Anthropicが5月にBlackstone・Hellman & Friedman・Goldman Sachsらと組成した$15億AI実装JVに、正式ブランド「Ode with Anthropic」が付与された。同社CEOでFractional AI共同創業者のChris Taylor氏はTechCrunchに「うまく実行すればいつか兆ドル企業を想像するのは簡単。課題はハイパーグロースで品質を落とさないことだ」と語る。BlackstoneがポートフォリオでのAI導入を大手コンサル・小型AIサービス会社に外注する中でFractional AIが際立ち、JV設立直後に買収（Fractionalは11か月続いたOpenAIとのパートナーシップを終了）。現在エンジニア100名を擁し、Anthropicの応用AIチームと密に連携して各企業の業務に合わせた実装設計を行う"スケール可能な高品質ブティック"を志向。既にOpenAIも同種の"The Deployment Company"を立ち上げており、フロンティアAIラボが"モデル出荷だけでは企業顧客を獲得できない"と認識する動きが加速。前日のTechCrunch「AI競争はフロンティアではなく実装層」分析を裏付ける主要ラボの構造転換で、コンサル業界（Accenture・Deloitte等）への正面攻撃としても機能する。

OnePlus、米国・欧州事業から今週撤退へ ─ 2026年スマホ出荷13%減予測(RAMageddon)、親会社Oppo再編の一環、インドも縮小、Realme欧州展開に集中

参照元: TechCrunch (Ivan Mehta) / Bloomberg / IDC / Counterpoint

中国スマホメーカーOnePlusが米国・欧州事業を今週閉鎖するとBloomberg報道。親会社Oppoの企業再編の一環で、インド市場（中国外の最大市場の一つ）も縮小予定。OnePlusは2013年にPete Lau・Carl Peiが低価格Androidスマホメーカーとして創業、その後拡張したがPeiは2020年にNothing創業のため離脱、フラッグシップの価格上昇と共にNordシリーズで低価格帯に回帰していた。ただし2026年のスマホ出荷は前年比13%以上減少するとIDC・Counterpoint予測、原因はメモリチップの供給制限＝業界通称"RAMageddon"（メモリ半導体スーパーサイクル・キオクシア数量断り報道と表裏）。Oppo自身もQ2に2桁台の出荷減、主要市場全般で需要軟化。OnePlusブランドは中国国内でのみ継続、海外はRealmeで北欧等の成功地域に絞る戦略。日経のメモリー"シリコンサイクル階段状"報道と同時進行で、"AIサーバー向けメモリ需要が消費者スマホを圧迫する"新しい半導体経済の副作用が具体化。

日本製鉄の収益力が急速に向上、粗鋼生産量1トンあたりの利益（EBIT/生産量ベース）で2026年1-3月期に世界の高炉大手首位となった。中国の過剰生産・鋼材需要低迷という厳しい事業環境下で、生産設備の再編（複数高炉停止）と価格交渉が奏功。日本経済新聞が世界主要高炉メーカーのEBIT・事業利益を粗鋼生産量で割ってトンあたり利益を算出し、USスチール（買収後の連結対象）除く日鉄本体が首位と確認した。中国粗鋼生産の減速・脱炭素対応での構造再編・自動車鋼板等の高付加価値化が鋼材メーカーの明暗を分ける中、"数量を追わずマージンで勝つ"日本製鉄モデルが世界標準の勝ちパターンとして浮上。同日 NippSteel USJ買収の統合効果も継続議論で、"日本の重工業再興"のシンボル案件として存在感を高める。同社株価にも中期的な業績改善期待が持続的に織り込まれる流れで、AI/半導体一辺倒の相場に対する"実物経済セクター"の反攻材料の1つ。

参照元: Federal Reserve Board / 時事通信 / 読売新聞 / ロイター / 第一ライフ資産運用経済研究所 (桂畑誠治)

Kevin Warsh FRB議長が7月14日、下院金融サービス委員会で就任後初の議会証言（半期金融政策報告・通称ハンフリー・ホーキンス証言）を約3時間実施した。冒頭で「委員会は高インフレの持続を容認しない・物価安定回復への断固たる決意」を表明、6月CPIの下振れ（前年比+3.5%）を"ガソリン下落という一時要因"として"任務完了とは全く考えていない"と警戒維持。中東情勢による原油高の"インフレ第2波懸念"については"外部要因を言い訳にしない・インフレは選択の問題"と断言。フォワードガイダンスは"認知バイアスを生む"として明確に排除、"データ・法律に厳格な会合毎判断"のレジーム・チェンジを鮮明化。AI関連投資の急拡大（機器投資+8%・ハイテク支出+25%）については"経済への恩恵の程度は不明だが、いずれ"AI投資"はただの"投資"と呼ばれる"と述べ、インフレ・雇用への影響を注視すると説明。デュアル・マンデートも"物価安定なくして雇用最大化なし"と実質シングル・マンデート寄りの立場。トランプ大統領の圧力には"全てを把握した上で扉を閉ざし最善の判断を下す"と独立性を強調。15日は上院銀行委員会で第2日目。市場は"7月末FOMCで政策変更なし"を再認識、AI CapEx依存の日米株式高との緊張が続く。

7月15日のアジア株式市場は急騰、KOSPI（韓国総合株価指数）は取引開始6分で買いサイドカー（プログラム買い注文一時停止）が発動、一時+7%超・7300台超まで急伸し3営業日ぶりに7000台を回復した。KOSDAQでも同時発動、今年36回目のサーキットブレーカー。Samsung Electronics +6%、SK Hynix +11%（前日ADRは+27%）で"米国株と韓国株の連動性強化"を示唆。日経平均は+1008円(+1.49%)高の6万8751円で終了、寄与度上位はアドバンテスト・東京エレクトロン・キオクシアHD・イビデンの半導体銘柄。オランダASMLが同日Q2好決算（売上・利益ともコンセンサス超え、AI半導体メーカー需要が中国懸念を相殺）を公表しレーザーテックが約2週間ぶりに5万円台回復。前日の米6月CPI予想下振れで9月利下げ期待再燃、SOX指数+2.5%超の反発を映した。逆にIBMショック連鎖でソフトウェア・コンサル関連は売られ"AIの恩恵セクター vs AIに代替されるセクター"の優勝劣敗が鮮明化。同日発表の米6月PPIも-0.3%（予想横ばい）・コア+0.2%（予想+0.4%）と二日連続のインフレ下振れで、Warsh議長のタカ派スタンスと市場の楽観の綱引きが続く。

SpaceX株、IPO価格$135割れ ─ 6/12 IPO後$200超からピーク時価総額200兆円消失、Starship打上げ16日直前、Anthropic・OpenAIのIPO見通しも黄信号

参照元: TechCrunch (Sean O'Kane) / 日本経済新聞

SpaceX株が7月15日、6月12日のIPO時公開価格$135を下回り一時$133台まで下落、その後$135を挟んで推移した。IPO直後は$200超まで急伸してAmazon・Microsoftに匹敵する時価総額を記録したが、日経報道では"ピークから時価総額200兆円消失"の水準に到達。ボラティリティの一因は"上場株数がわずか4%のスモールフロート"と"過剰な注目"の組合せ。IPOで調達した債券価格も下落中で、"マスク氏の壮大なビジョンに市場が冷めつつある"広範な下半期テック株調整の一部と分析される。16日にはStarshipの試験打上げがIPO以降初めて実施予定（5月ブースター故障以降）、失敗リスクを持つ"fly, fail, fix"アプローチ継続、ブースター・上段の回収は行わない。SpaceX株はAnthropic・OpenAIの秘密IPO申請の"市場テスト"の役割を果たすため長期低下は他フロンティア企業の上場動向にも影響。前日Meta Mosseriのトークン予算論・IBMミス・Anthropic広告炎上・DeepSeek IPO計画と併せて、"フロンティアAI・宇宙のIPO熱の折り返し"が観測される可能性がある転換点。

前日からの続報ニチレイ、個人情報漏洩の可能性を発表 ─ サイバー攻撃の被害波及、KFC・イオン・くら寿司などに配送影響拡大、大手食品物流の供給網リスクが日本経済の"新しい急所"に

参照元: 時事通信 / 共同通信 / 日本経済新聞 / ニチレイ

冷凍食品・食品物流大手ニチレイは7月15日、前日発覚したシステム障害に関し"個人情報が漏洩した可能性がある"と発表した（詳細範囲は追加調査中）。被害の影響は日本KFCの一部店舗の食材調達停滞・臨時休業のリスクに続き、イオンやくら寿司にも配送影響が拡大していると時事通信が報じた。ニチレイは食材の一次卸・冷凍物流を担う基幹インフラで、外食・小売・給食が広く依存する構造が明らかに。共同通信も個人情報漏洩の可能性を追加で確認、"ランサム型を含む詳細"は引き続き調査中。前月のアサヒビール系サイバー攻撃、ニデック等の会計問題と併せ、日本の"生活インフラ企業"のサイバーレジリエンスへの警告が続く。同日サイバーセキュリティ対策「AI計画、技術革新受け改定」の政府方針が公表されたことも符合。前日のIBM Q2ショックの背景（顧客企業のサイバーセキュリティ懸念によるIT支出配分の混乱）を日本国内で裏付ける第2の事例となり、東証食品セクター・物流セクターの見直し材料に。

本日最大米国、新対ロ制裁法案 ─ 中国・インドなど"ロシア産原油購入5カ国"に最大100%関税、トランプ大統領支持で上院可決へ、"イラン・ヒズボラも対象"と発言、日本・フランスは天然ガス輸入比率15%未満で対象外

参照元: Bloomberg / TBS CROSS DIG with Bloomberg / dメニューニュース

米共和党の故リンゼー・グラム上院議員が主導した対ロシア制裁法案が浮上、中国・インドを含むロシア産原油・天然ガス購入の主要5カ国が対象で、最大100%の関税を課すことができる内容と7月15日報道。ウクライナ侵攻終結へロシアへの経済圧力を強める狙いで、最終的関税率は米通商代表部（USTR）が"ロシア産原油購入を抑制できる水準"で決定する見通し。上院スタッフによると、法案提出者らはトランプ大統領支持を得たため上院可決に十分な支持があるとみるが採決時期は不透明。トランプ氏は同法案の対象にイラン・ヒズボラも加わる可能性を発言、"非常に大きなこと"と表現。ロシア産天然ガス輸入比率15%未満の国は適用除外で、日本・フランスは対象外となる見通し。100%関税発動なら既に不安定な米中・米印貿易関係にさらなる影響を及ぼす可能性があり、ホルムズ海峡・ウクライナ戦況・湾岸攻撃と並ぶ"下半期の地政学トリガー"の一つに。前日の米イラン攻撃応酬・トランプ通航料撤回と併せ、通商政策の同時多発は市場が最も警戒する不確実性の増幅要因。

高市首相、党首討論で"骨太ショック原因でない"と否定 ─ "強い経済"重視・「成長のスイッチ押しまくる」、玉木代表「気合と根性でマーケットは動かない」、消費税減税結論は"8月頭"

参照元: 日本経済新聞 / 時事通信

高市早苗首相が7月15日の党首討論（今国会2回目、45→60分に拡大）で、6月30日の「骨太の方針」原案が長期金利上昇・円安を引き起こした"骨太ショック"の原因ではないとの見方を表明した。国民民主党・玉木雄一郎代表の質問に「閣議決定もしていない原案がショックの原因だと思っていない」「為替市場や金利は様々な要因で決まる」と反論、「強い経済」の重要性を説き「成長のスイッチを押して、押して、押しまくる」と強調。玉木代表は「気合と根性だけでマーケットは動かない」と応じた。中東情勢対応の予備費使用については「必要なときはちゅうちょなくタイミングを逸せず対応する」と述べ、消費税減税を検討する国民会議の結論は「8月冒頭であれば間に合う」と初めて具体的時期を表明。参政党・神谷宗幣代表がスパイ防止法に関連して国家監視機関設立を求めたのに対し「通信の秘密は憲法上の基本的人権」と慎重姿勢。前日20年債入札好調・GPIF国内投資拡大観測に続く"官製相場化" vs "市場の警鐘"の綱引きの第2ラウンドで、政府は21日に骨太方針最終案（日銀の自主性尊重明記）を閣議決定予定と併行して展開。

改正金融商品取引法が可決成立、暗号資産を"金融商品"に再分類 ─ 現物ビットコインETFへの道開く、無登録業者の罰則を懲役3年→10年に、2028年から所得税最大55%→20%一律に

参照元: NHKニュース / CoinDesk日本版 / 時事通信

日本の参議院本会議は7月15日、暗号資産を"金融商品"と位置付ける改正金融商品取引法・改正資金決済法を可決成立させた（施行は2027年）。従来の"支払い手段"を主眼としたPSA（資金決済法）中心の枠組みから、他の金融商品と同様に"投資商品"として扱う枠組みへの構造的シフトで、将来の現物ビットコインETFに対する重要な法的障壁を除去した（ETF商品の承認自体は行われず、金融庁が今後規制枠組みを検討）。無登録暗号資産事業者の最高刑期は懲役3年→10年、最高罰金は300万円→1000万円に強化、インサイダー取引規制導入、発行者・取引所への定期開示義務・投資家保護要件も拡大。所得税は2028年から現行最大55%→20%一律（国15%+地方5%）に引き下げ枠組みを承認、昨年末に政府・与党連合が主導し導入した提案が実現。世界の主要国が暗号資産ETF承認を進める中、日本が"税制優遇＋規制強化"の両輪で追随する形で、暗号資産市場・フィンテック・機関投資家の資産配分に中期的影響。同日 SBI・GMO等の日本仮想通貨関連銘柄の中期的トリガーに。

2026年上半期の訪日客、5年ぶり前年割れ ─ 中国渡航自粛が響く、対日レアアース輸出も低水準継続、日中関係悪化がインバウンド経済の変数として再浮上

参照元: 時事通信 / 日本政府観光局(JNTO) / Bloomberg

2026年上半期（1-6月）の訪日外国人数が5年ぶりに前年同期を下回ったと7月15日発表、主因は中国からの渡航自粛。高市首相の昨年台湾関連発言に対する中国政府の反発以来、日中関係悪化が続き、1月から中国政府はデュアルユース品の対日輸出規制・審査を強化。5月中国から日本への重要鉱物輸出のうち、タングステン主要種の一部・レアアース（ジスプロシウム、テルビウム）はゼロ、他のレアアース品目も異例の低水準（Bloomberg・中国税関データ）。イットリウムも過去3か月移動平均で2023年以来最低水準。G7エビアン首脳会議は"レアアース単一国供給比率を2030年までに60%超えないこと"で合意したが具体的政策枠組みは未合意。円安・原油高でインバウンド収支が支えていた日本経済にとって、"消費・観光"と"素材調達"の二重リスクが同時進行する構図。円相場は同日午後5時時点で162円26-27銭と依然安値圏を推移。政策関係者は"訪日客が中国依存を続ける限りチャイナリスクは経済安全保障の裏面"と指摘、多国籍化戦略の再検討を促す。
