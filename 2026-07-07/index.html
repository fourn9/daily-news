本日最大サムスン電子、4-6月営業益89.4兆ウォン（＝約9.3兆円）で前年同期比19倍 ─ AIメモリー急騰でNvidiaの四半期益超え、ただし株価は6.8％急落

参照元: CNA (Reuters) / Korea Herald / The Elec / TNW / みんかぶ（Techmemeトップ集約）

サムスン電子が7月7日、2026年Q2（4-6月）の暫定営業利益89.4兆ウォン（US$58.44B＝約9.3兆円）を公表、前年同期比19倍、LSEG市場予想87.3兆ウォンも上回った。売上は前年同期比+129%の171兆ウォンで、いずれも四半期として過去最高。四半期営業益ではNvidiaが今年2月に記録した$53.5Bも超え、世界最大級の四半期利益企業に浮上。HBMのAIサーバ向け需要が汎用DRAM／NAND価格まで押し上げ、DS部門の営業益は80兆ウォン後半に達したもよう。ただし利益確定売りとAIインフラ投資ピーク観測でサムスン株は開場後-6.8%急落、韓国KOSPI下落を通じて東京市場のAI・半導体株も連鎖安に（日経-1480円）。「メモリーのスーパーサイクルは構造的か景気的か」が改めて焦点。

Meta、Superintelligence Labsから初の画像生成モデル「Muse Image」投入 ─ Instagram/WhatsApp/広告に一斉統合、Alexandr Wang氏率いる旗艦成果物

参照元: TNW / Meta AI blog / Techmeme集約

Metaが7月7日、Alexandr Wang氏率いるMeta Superintelligence Labs（MSL）が開発した初の画像生成モデル「Muse Image」を発表。Instagram、WhatsApp、Facebook、および広告プラットフォームで即日展開。4月にリリースしたテキストモデル「Muse Spark」に続くMuseファミリー第2弾で、MSLがWang氏を中心に再編されて以降の主要プロダクトの初出荷となる。広告主向けにはクリエイティブ自動生成、消費者向けにはチャット内画像生成として組み込まれ、既存の外部モデル依存（過去はStable Diffusion系）からの独立を鮮明にする戦略。OpenAI GPT-4o Image、Google Nano Banana、Adobe Firefly 4と競合し、月間ARPU拡大が経営目標。

ECB＋ESRB、フロンティアAIを「金融システムのシステミックリスク」に指定 ─ ユーロ圏銀行に10月31日までの対策計画提出を要求

参照元: Reuters (Global Banking) / Financial Times / ESMA / Regulation Tomorrow（Techmemeトップ集約）

欧州中央銀行（ECB）は7月7日、ユーロ圏の銀行CEO宛書簡で、フロンティアAIが「情報通信システムの機密性・完全性・レジリエンスに深刻な影響を及ぼしうる」として、AI起点サイバー攻撃への対策計画を10月31日までに提出するよう要請。同日、欧州システミックリスク理事会（ESRB）が「フロンティアAIモデルは金融システムのシステミックリスクの源泉」と警告する初の公式声明を出し、EBA/EIOPA/ESMAの3局（ESAs）も支持を表明。想定シナリオには「小規模銀行への信認喪失連鎖」「国家関与のスパイ活動」「決済・清算システムへの協調攻撃＋偽情報増幅」が含まれ、フロンティアAIが「数分から数時間」でIT脆弱性を突けるとの指摘が土台。DORA/AI法の既存枠を「加速化」する形で、規制産業のAI導入・防御に4か月の時間軸が突きつけられた。

Anthropic、Claude内部に「J-space」構造を発見 ─ グローバルワークスペース理論と酷似、Jacobian lensで多段推論を可視化しプロンプトインジェクション検知も

参照元: Crypto Briefing / Anthropic (transformer-circuits.pub) / Techmeme集約

Anthropicの解釈可能性チームが7月6日、Claude Sonnet 4.5の内部に「J-space」と名付けた共有ワークスペース構造を発見したと発表、7月7日に外部有識者コメントとともに報道が本格化。神経科学のGlobal Workspace Theory（Baars, Dehaene）に酷似した「共有黒板」に情報を書き込み・読み出しながら中間推論を保持する仕組み。開発されたJacobian lensで観察可能で、J-spaceを無効化すると単純タスクは処理できても複雑な多段推論が崩壊。副次的にはプロンプトインジェクションや幻覚（fabricated data）を出力前に検知できる「安全弁」としても機能。「AIモデルは単なるパターン照合ではなく、動的な内的表現を保持している」ことの初の機械的証拠として学界に波紋。実装はオープンソース化。

ヤマダHD×エディオン、経営統合基本合意 ─ 「対等の精神」で27年10月に共同持株会社、久保会長CEO「日本の家電市場は右肩下がり」と危機感

参照元: 読売新聞 (dメニューニュース経由)

エディオンの久保允誉会長CEO（76）が6日、読売インタビューで、家電量販首位ヤマダホールディングス（HD）との経営統合について「対等の精神で協議を進めた」ことが合意の決め手だったと明かした。両社は2027年10月に持株会社を新設し完全子会社化する方針。新会社の会長はヤマダの山田昇氏、社長は久保氏で共同CEO、取締役も同数選任。「日本の家電市場は右肩下がり、異業種参入も激しい」との危機感が背景で、PB商品の量的メリットと物流統合を狙う。統合効果の実現には「7-8年」を要すると発言。筆頭株主ニトリHDの動向が焦点で、家電・ホームセンター・EC横断の日本流通再編の号砲となる可能性。

ソニー銀行、米OCCから信託子会社設立の条件付き承認 ─ 2027年度にドル建てステーブルコイン発行、ソニーゲーム/アニメ課金の"内製決済化"へ

参照元: 日本経済新聞 (Digital Asset Lab経由)

ソニー銀行が米通貨監督庁（OCC）から、米国内での信託子会社「コネクティアトラスト」設立について条件付き承認を7月6日取得。正式承認後の2027年度にドル建てステーブルコインを発行し、ソニーグループのゲーム・アニメ課金の決済インフラに組み込む計画。クレジットカード手数料（数%）の削減が主目的で、Circle、Rippleなど十数社に続く邦銀初級の動き。国内では2023年施行改正資金決済法でステーブルコインが法的に定義されており、米国でノウハウを積んだ後の国内展開も視野。日本の銀行が米国でステーブルコイン事業に本格参入する意義は大きく、Web3インフラの主導権争いに邦銀・大手プラットフォーマー連合が名乗り。

参照元: 時事通信 / 毎日新聞 / TBS NEWS DIG (Bloomberg) / NHK

7日の東京債券市場で、10年物新発国債利回りが一時2.850％まで上昇し、1997年5月以来約29年ぶりの高水準を更新（前日は2.830%）。米金利上昇に加え、日本の財政悪化・インフレ進行への警戒が加速。城内実経済財政担当相は7日の会見で、6/30に公表された「骨太の方針」原案が日銀の利上げをけん制していると受け止められている件について「趣旨と異なる誤解」と釈明、「政府が低金利誘導を促している事実は全くない」と述べたが、市場の警戒感は解けず。大手証券は「買い材料が見当たらず、長期金利は3％を超える可能性もある」と発言。骨太決定は7月中の見通しで、8日の30年債入札の需給が次の耐久試験。

国内政治自民・維新、社会保障改革の骨子で合意 ─ 70歳以上医療費「原則3割」は記さず、年末までに工程表、外来特例・年齢区分見直しへ

参照元: 日本経済新聞 / 毎日新聞

自民党（田村憲久元厚労相）と日本維新の会（梅村聡税調会長）が7日、社会保障制度改革の骨子で合意、国会内で署名。70歳以上の医療費窓口負担引き上げに向けて2026年末までに工程表を策定するが、維新が求めた「原則3割」への引き上げ明記は見送り。骨子は「原則3割となっている現役世代との間で年齢によらない真に公平な応能負担を実現する観点から見直す」と表現し、①外来特例、②所得区分、③年齢区分の見直しを3本柱に。同時に第3号被保険者制度の対象縮小、医療機関の消費税負担軽減、大学病院機能強化も盛り込む。7月内に閣議決定される「骨太の方針」に反映される見通しで、「原則3割」を巡る自維の温度差は年末工程表策定時に再燃必至。
