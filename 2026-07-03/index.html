本日最大アリババ、社内でClaude Code禁止 ─ Anthropicの「中国ユーザー追跡コード」発覚、7月10日から

参照元: South China Morning Post / Reuters / The Next Web / Caixin Global（Techmemeトップ集約60媒体超）

アリババが社内文書で「Claude Codeにバックドアリスクが確認された」として高リスクソフトウェアに指定、7月10日から全社員に利用を禁止、自社の「Qoder」への切替を通知。Reddit投稿者LegitMichel777が6月30日に公開した解析で、Claude Codeがバージョン2.1.91（4月2日リリース）以降、Asia/Shanghai等のタイムゾーンやプロキシURLをステガノグラフィで検出する隠しコードを含んでいたことが判明。Anthropicのエンジニアは「不正リセラーとdistillation防止の実験」と釈明、7月1日に削除PRをマージ済みと述べたが、6月10日にAnthropicが上院委員会宛書簡でアリババQwen関連の2.5万偽アカウント・2,880万往復のdistillation攻撃を指弾した経緯があり、米中AI冷戦の象徴的事件に発展。

Meta「Watermelon」がGPT-5.5に追いつく ─ Alexandr Wang氏がタウンホールで従業員に報告

参照元: Business Insider (via AOL) / InfoWorld / Yellow.com / TechCrunch

Meta Superintelligence Labsのアレクサンドル・ワン氏が7月2日の社内タウンホールで、現在訓練中の次期基盤モデル「Watermelon」が主要ベンチマークでOpenAIのGPT-5.5に追いついたと報告。前世代Avocado（Muse Spark）比で「一桁多い演算量」を投入。同氏はX投稿で「Muse Spark更新版もコーディング・エージェント能力で近日改善」とし、Anthropic Claude Opus級コーディング能力の時期を「近い」と回答。今年AIインフラ1,450億ドル投資を続けるMeta。ザッカーバーグCEOはAIエージェントの進化が想定より遅いとも認めており、対抗馬到達を賭ける勝負所。

ザッカーバーグCEO「AIエージェントは期待ほど加速していない」 ─ 8,000人リストラも成果出ず

参照元: TechCrunch (Lucas Ropek) / Reuters

ザッカーバーグ氏が7月2日の社内タウンホールでAIエージェント開発の進捗が幹部の想定通りに「加速していない」と表明。今年実施した約8,000人のリストラ（法人従業員の約10%）と7,000人のAI部門再配置を「クリーンでなかった」と振り返り、AI再編の上振れは「まだ実を結んでいない」と発言。3〜6ヶ月で改善が見えると強調するも、Meta AI部門は現場エンジニアから「魂を削る強制収容所」と評され厳しい状況。今年AIインフラ最大1,450億ドル投資が正当化されるかが問われる局面。

Godot財団、AI生成コード受入を禁止へ ─ 「vibe-coded」PR氾濫でメンテナー疲弊

参照元: The Register / Game Developer / Godot Engine公式ブログ

オープンソースゲームエンジンGodotの財団が6月30日、AIエージェント経由のPRおよびAIによる大規模コード生成を明文で禁止する方針を発表。「自律AIエージェント／vibe codingは即GitHubからBAN」「補完・正規表現・置換など menial なAI利用に限定」「PR議論での機械生成テキストも禁止」など厳格化。AIに責任は取れず、AI多用者はコードを理解せず修正できない、と説明。オープンソース開発者コミュニティ全体で「AI slop PR」の是非を巡る議論を加速させる可能性大。

Sysdig、初の「エージェント型ランサムウェア」JadePufferを確認 ─ LLMが自律的にDB暗号化

参照元: SecurityWeek / The Register / Sysdig公式ブログ

SysdigのThreat Researchチームが、初めて確認されたエージェント型ランサムウェア攻撃「JadePuffer」を報告。攻撃者はLangflow(LLM開発フレームワーク)のCVE-2025-3248脆弱性を突いてアクセス、LLMがAPIキー・クラウド資格情報・仮想通貨ウォレット・DB認証情報を自律的に列挙。標的の本番MySQL/Alibaba Nacos設定サービスへピボットし、認可バイパス(CVE-2021-29441)と署名鍵デフォルトを悪用、1,342項目を暗号化。攻撃者は身代金要求だけ残し、暗号鍵は保存されず復旧不能。「31秒でエラーから修復コード生成」など人手速度を超え、AIが攻撃者ツールキットを置き換える時代の到来を告げる事例。

シボレーの「全米製EVトラック」なぜ売れない ─ 関税でBOM有利も需要低迷、市場設計を問う

参照元: TechCrunch (Tim De Chant)

GMがアメリカ国内サプライチェーンで組み立てる電気ピックアップトラック「Silverado EV」の販売が想定を大きく下回っている構造を分析。トランプ122条関税で輸入部品コストが上がる中、国内生産BOMが相対的に有利になっているにもかかわらず、EVピックアップ市場そのものが縮小しており米EV政策の効果に疑問符。給油スタンドから充電への切替コスト、モデルレンジ設計、価格帯の問題が浮上し、米自動車政策と関税政策の整合性が問われる展開。

続報長期金利、一時2.81% 30年ぶり高水準 ─ 「骨太ショック」で秋には3%視野の警戒感

参照元: 日本経済新聞 / 時事通信 / 読売新聞

(前日からの続報) 3日の東京債券市場で新発10年物国債の流通利回りが一時2.810%まで上昇、1996年10月以来約30年ぶりの高水準。高市政権が6月30日に示した「骨太の方針」原案で「日銀の適切な金融政策運営が非常に重要」と明記されたことが、日銀利上げけん制と受け止められ債券売りが加速。三井住友トラスト・アセットの稲留氏は「拡張的財政の方針が変わらなければ秋に3%に達する可能性」と警告。米欧金利は落ち着く中、日本固有の財政・金融政策懸念で金利上昇が続く「骨太ショック」との呼び方も登場。片山さつき財相は「責任ある積極財政」を強調するも市場との対話が急務に。

参照元: 日本経済新聞

米労働省が7月2日発表した6月雇用統計は非農業部門雇用者数が前月比+5.7万人と大幅未達、失業率は4.2%に微上昇。W杯開催地の労働需要は限定的で、市場が期待した「W杯効果」は現れず、労働市場の減速を鮮明化。ただし労働参加率は61.5%まで低下(Covid期を除きprime age層の縮小は1976年以来)、720万人が労働力から離脱している構造も浮き彫りに。米国内では早期利上げ観測が後退、ドルが1ドル=161円台まで急速に円高、NY円急伸で日本の介入ラインを瀬踏みする展開。

参照元: TechCrunch (Zack Whittaker) / Citizen Lab
