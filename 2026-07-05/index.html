本日最大OpenAI「Stargate UK」£20B構想は「政府PRの空約束」だった ─ Guardianスクープ、Cobalt Park現地訪問ゼロ

参照元: The Guardian (Aisha Down) / The Next Web / Techmeme集約

Guardianが7月4日、英政府とOpenAIが2025年に発表した英国最大級のAI投資計画「Stargate UK」の実態を暴露。目玉だった北Tyneside・Cobalt Parkの£20B（約4兆円）データセンター構想について、情報公開請求で「OpenAIもパートナーのNscaleも一度も現地自治体に接触していなかった」と判明。訪問実績があったのはNvidiaだけ（トランプ訪英から5か月後の2026年2月）。関係者は「Nscaleは事前通知なしにStargate参画を強要された。実質は政府のPRスタント。プラグが抜かれたときSam Altmanが責を負っただけ」と語る。政府が「£30B投資予定」と宣伝したうちの£20Bは"仮想"、実際に契約済みだったのはBlackstoneの£10Bのみ。3月のGuardian報道「AIを英経済の血管に注入する取引の多くは幽霊投資」の続編で、英米AI協力の看板が虚構だったことを示す一撃。

Godot Engine財団、AI生成コード寄稿を全面禁止へ ─ 「AIは責任を取れず、ヘビーユーザーは自分のコードを理解していない」

参照元: PC Gamer / The Register / Kotaku（Slashdotトップ集約）

オープンソースのゲームエンジン「Godot」（Slay the Spire 2などで採用）を運営するGodot Foundationが、コントリビュータ規約を改定し「全コードは人間著者であること」を必須化。AI生成のPR・自律エージェント提出・AI生成の議論文書までも禁止対象で、違反はGitHubリポからの自動BAN。既に5,000超の未解決PRが山積みで、多くがAI生成疑い。マージPRが3件以下の新規コントリビュータには機能追加・大規模リファクタは事前許可制。「AIは責任を負えず、AIヘビーユーザーは自分のコードを理解しておらず修正依頼にまともに応答できない」「レビュアーの労力を機械に吸収させるのは基本的な敬意の欠如」との説明。VibeコーディングOSS流入の弊害を認めた初の大型プロジェクトで、他OSSプロジェクトの方針にも波及するかが焦点。

円買い介入の効果、FX「ミセス・ワタナベ」が減殺 ─ 巨額円買いポジションの利益確定売りで円高が続かず

参照元: 日本経済新聞

円安防止のため為替介入の機会をうかがう政府・日銀にとって、FX個人投資家（通称ミセス・ワタナベ）の巨額円買いポジションが新たな悩みの種に。介入時の利益確定を狙って前もって円買いを異例の規模に膨らませており、円買い介入で円が上昇した瞬間に一斉に円売り戻しが入る構造。介入効果を減殺するだけでなく、日銀の政策後手観測とも重なりドル円は161円台で膠着。片山さつき財務相は米財務省と「断固たる措置」で連携する方針を表明しているが、個人投資家経由の"介入前予測売買"が円安是正の実質的な壁として立ちはだかっている構図が浮き彫りに。

ウクライナ、サンクトペテルブルクの大型石油ターミナルとバルト艦隊基地を無人機攻撃 ─ 850km長距離飛行、露北西部で最大規模の一撃

参照元: BBC / Al Jazeera / Moscow Times / Kyiv Independent（Nikkeiも報道）

7月4日夜、ウクライナ軍の長距離無人機がロシア第二の都市サンクトペテルブルクを大規模攻撃、Kirovsky地区の石油ターミナル（年間1,250万トン処理能力、"露最大級")を破壊。同時にKronstadtのバルト艦隊主要基地も攻撃、Peterhof宮殿敷地内にも一部落下。国境から850〜900kmという同市への攻撃としては過去最深遠。露は「地域上空で72機を撃墜」と発表するが、ゼレンスキー氏は動画で黒煙を公開。ウクライナ参謀本部は「露石油精製能力の42.7%を無効化、8か月で60超のタンクを破壊、露産業損失$13.5B」と主張。露国内では40超の地域でガソリン販売制限・輸出禁止延長が続き、プーチン氏も「深刻ではないが不足は認識」と発言。長距離ドローン戦のフェーズが「露首都圏聖域化の終焉」段階へ移行。

マイナンバー活用、政府が国民に直接現金給付 ─ 「デジタル敗戦」教訓、感染症・災害時に自治体を通さない給付基盤へ

参照元: 共同通信 / 読売新聞 / d menuニュース

政府がまとめるデジタル社会重点計画2026年版が7月2日に判明、Nikkei7月5日朝刊でも要旨掲載。マイナンバー・公金受取口座を活用し、政府（デジタル庁+日銀+ADAMSシステム）が自治体を経由せず直接国民に現金給付できる基盤を整備する方針。コロナ禍で市町村に負荷が集中した「デジタル敗戦」を教訓化。同重点計画にはスマホに免許情報を格納する「モバイル運転免許証」実現も明記。ただし公金受取口座登録率は依然約5割、100%達成が課題。29年度に予定される中低所得者向け新給付制度も口座登録前提で検討。国民ID・給付・免許のデジタル完結という日本行政の"最終形"への地ならしが本格化。
