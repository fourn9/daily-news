本日最大Microsoft、Xboxで3,200人（全体20％）削減＋5スタジオ切り離し ─ "史上最大の再編"、Ninja Theory/Undead Labs売却

参照元: Bloomberg (Jason Schreier) / The Verge / Windows Central / Game File（Techmemeトップ集約）

MicrosoftのXbox新CEO Asha Sharma氏が7月6日、社員宛メールで「Xbox史上最大の構造改革」を発表。FY27末までに約3,200人（Xbox全体の約20％）を削減、うち1,600人は即日、残り1,600人は今後12か月で削減。同時に5つの傘下スタジオを切り離し ─ Compulsion Games／Double Fineは独立化、Ninja Theory／Undead Labsは売却、"Senua"シリーズのArkaneも協議入り。Microsoft全体でも約4,800人（総従業員2.1％）が対象で、大半が営業とXbox。2024年に9,100人・2025年にも数百人を削っており、Xbox買収戦略から「利益重視」への軌道修正が鮮明に。ゲーム業界史に残る"リセット"。

Nvidia、次世代AIラック「Kyber NVL144」を12か月以上遅延し2028年に延期 ─ 姉妹アーキテクチャNVL72x2は開発中止（SemiAnalysis）

参照元: SemiAnalysis / CNBC (Anniek Bao)（Techmeme集約）

SemiAnalysisが7月6日、NvidiaのRubin Ultra世代（2027年）向け次期ラック規模アーキテクチャ「Kyber NVL144」が、PCB（プリント基板）製造の課題により12か月以上遅延し2028年に後ずれすると報道。同時に姉妹となるNVL72x2アーキテクチャは開発中止に。3月のGTCでJensen Huang CEOが華々しくデモしてからわずか3か月での大幅スリップで、AIインフラのロードマップに影響。Kyberは液冷・電力・光通信の統合を前提とした次世代設計だっただけに、Rubin Ultra世代のGPU供給スケジュールも間接的に影響を受ける可能性。ハイパースケーラーのAI設備投資計画にとって痛手。

「JadePuffer」─ 世界初のエージェンティックランサムウェアをSysdigが観測、LLMが単独でシステム侵入から身代金要求まで完遂

参照元: CSO Online (Gyana Swain) / Digital Trends / BleepingComputer / TNW（Techmemeトップ集約）

クラウドセキュリティ企業Sysdigが7月6日、世界初と評価する「完全にAIエージェント駆動のランサムウェア攻撃」を観測したと発表。「JadePuffer」と命名された攻撃は、公開LangflowサーバのCVE-2025-3248を突破点に侵入、認証情報収集・ラテラルムーブ・Alibaba Nacosの1,342件のDB設定を暗号化、身代金要求まで600超のペイロードをLLMが自律的に発行。失敗時に31秒でリカバリするなど、これまで熟練攻撃者の領域だった動的適応をAIが代替。ランサムノート内Bitcoinアドレスがドキュメント例示のものという痕跡から、AI生成である証跡も確認。「エージェンティック脅威アクター時代の到来」を告げる象徴的事案として、CISOたちの警戒レベルが一段引き上げに。

Broadcom、Appleとのカスタムチップ供給契約を2031年まで延長 ─ AI/5G向け半導体で数十億ドル規模の複数年契約

参照元: Reuters (Akash Sriram)（Techmeme集約）

Broadcom（AVGO）が7月6日、Appleとの戦略的パートナーシップを2031年まで延長し、カスタム設計チップの共同開発・供給を継続すると発表。2023年に発表された数十億ドル規模の複数年5G部品契約の拡張版で、AppleのA/M/Sシリーズ内部の無線・ネットワーク・光関連コンポーネントを長期供給する構造。Apple自社設計チップ内製戦略のなかでBroadcomが「離せない相棒」であることを再確認する形。前週のAppleサプライヤー動向（Ming-Chi Kuo氏「foldable iPhone Q3出荷100万台未満、発売Q4へ後ずれ」報道）と合わせて、Appleサプライチェーン全体の再編が続いている。

日産サクラ、国＋東京都補助で実質56万円 ─ 4-6月EV販売シェア初の3%超え、"補助金が市場を歪める"の指摘

参照元: 日本経済新聞

7月6日公表の自動車販売連盟データで、2026年4-6月の国内EV販売台数は前年同期比2.7倍の3万2,378台、新車全体に占める比率は四半期として初めて3%を突破。背景には国と東京都の両補助金（都単独で最大130万円、国と併用可）があり、日産サクラは実質56万円まで下がる異例の水準に。100万円を割る車種が複数出て需要を刺激する一方、記事は「補助金が市場を歪めている」との論調。トヨタが25年10-12月にEV販売首位に浮上した局面と重なり、日本のEV政策の"出口設計"（補助金終了後の需要）が課題として鮮明化。都のEV補助はテスラ、トヨタなど輸入・国産大手に恩恵が偏る形になっている。

サッポロビール、カールスバーグと提携 ─ 東南アジア市場強化、少子高齢化に伴う国内縮小への出口戦略

参照元: 時事通信 / dメニューニュース

サッポロホールディングス傘下のサッポロビールが、デンマークの世界大手カールスバーグと東南アジア市場での戦略的提携を7月6日発表。ベトナム・タイ・シンガポールなどでの生産・販売・物流を相互補完し、両社ブランドを効率的に展開する枠組み。人口減少で長期縮小する日本市場から成長市場へのシフトを本格化させる動きで、キリン・アサヒが先行して海外M&Aを進めてきた業界構図に対する追随策。円安局面で海外事業の連結収益の伸長も期待され、株式市場は好感（サッポロHD株が同日上昇）。

参照元: TechCrunch (Kate Park)

中国発のスマートグラス「Even Realities」が7月6日、Meituan（美団）とTencent（騰訊）主導で$150Mを調達、評価額$1Bのユニコーンに。同社は透明ディスプレイ搭載のミニマルデザイン眼鏡「G1」等で、Meta Ray-BanやXreal、Xiaomi Smart Glassesがひしめく分野の中で「デイリー使いできるファッション性」で差別化。Meituanは配達員向け・料理検索AR、Tencentは翻訳・通知連携などスーパーアプリ側の統合を狙う。中国スマートグラス市場は26年で年間出荷1,000万台超に迫る勢いで、Physical AI/AR時代の消費者インターフェース覇権争いにおいて中国連合が本格参戦する象徴的ラウンド。

上海Biren、追加株売却でHK$70億（約$892.5M）調達 ─ 中国AI GPU国産化を加速、香港上場後株価150%超

参照元: South China Morning Post (Ann Cao)（Techmeme集約）

上海拠点のAI/GPUチップメーカー壁仞科技（Shanghai Biren Technology）が、香港市場でHK$70億（約$892.5M＝約1,320億円）の追加株式売却を実施。GPU増産と量産能力拡大に充てる。同社は今年1月に香港でIPOし、以降株価が150%以上上昇するなど中国AIチップ国産化の旗艦銘柄となっており、米国の先端GPU輸出規制の中で華為・寒武紀・海光と並ぶ有力供給元に。今回の資金調達は、中国政府系ファンドと戦略投資家の追加コミットに支えられ、NvidiaのKyber延期（本日別記事）とも相まって、中国AIハードウェアの独自路線が加速する構図が浮き彫りに。

銀行AIエージェント基盤のTaktile、Goldman Sachs主導で$110M調達 ─ 「銀行のエージェントファースト化」を主導

参照元: PYMNTS

ベルリン発のAIエージェント基盤スタートアップ「Taktile」が、Goldman Sachs Alternativesリードで$110Mの資金調達を発表。銀行・保険・フィンテック向けに、審査・不正検知・オペレーションの自律判断エージェントを、規制遵守と監査可能性を担保して提供するプラットフォーム。共同CEOのMaximilian Eber氏は「金融機関の複雑業務は今後2年で急速にエージェント化する」と発言。Anthropic・OpenAIのフロンティアモデル、および銀行内部のリスクポリシーを組み合わせた"制御可能なエージェント"が売り。銀行AIエージェントを巡ってはSalesforce、Palantir、Cursor共同創業者らも動いており、規制産業向けAIの主戦場として資金流入が加速。

SK Hynix、米Nasdaqで約$28B調達の株式売出開始 ─ 1,779万株、HBM需要拡大投資、金曜からNasdaq取引開始

参照元: Reuters（Techmeme集約）

SK Hynixが7月6日、米NasdaqでのUS存託証券（ADR）上場を伴う株式売出（約$28B、1,779万株）を開始。ソウル本社上場に加え米国での資本調達手段を確保する狙いで、Nvidia向けHBM3E/HBM4供給拡大に向けた設備投資に充当。最終価格は木曜決定、金曜から取引開始予定。米中AI競争の中でHBM市場をほぼ独占するSK Hynixにとって、AI hyperscalerとの直接的な資本市場アクセスが戦略的意味を持つ。同社株はソウル市場で年初来+40%超と、AI関連のなかで最も物色されている銘柄群のひとつ。

日本市場日経平均6.38円安 小反落だがTOPIXは6連騰で史上最高値 ─ AI・半導体からバリュー株へのローテーション鮮明化

長期金利、一時2.830%まで上昇 ─ 29年ぶり高水準、「骨太ショック」で財政悪化懸念売り

参照元: 時事通信

7月6日の東京債券市場で、10年物新発国債利回りが一時2.830%まで上昇し、1997年5月以来約29年ぶりの高水準に。日本相互証券が公表。背景は6月末に高市政権が示した「経済財政運営と改革の基本方針（骨太の方針）」原案で、従来盛り込まれていた「財政健全化」の文言が削除されたこと。市場は「大型財政出動＋防衛費増額＋日銀正常化継続」の組み合わせを警戒し、国債売りが加速する「骨太ショック」の様相。30年ぶり水準の長期金利上昇は、変動金利型住宅ローン借入者や、日銀保有国債の含み損拡大を通じて日銀財務にも波及するリスクを孕む。8日から30年債入札が予定され、需給の耐久力が試される。

歴史的円安162円台、家計負担年1.6万円増との試算 ─ 円安倒産は上半期45件で前年1.3倍、中小に逆風

参照元: 時事通信 / みずほ総合研究所 / 東京商工リサーチ

ドル円が7月1日に162円80銭台と1986年12月以来39年半ぶりの安値を更新、その後も円安圧力が持続。みずほ総合研究所の試算では、1ドル=162円で1年間推移すると家計負担は25年比で平均1万5,534円増、収入300万円未満世帯は9,878円増、1,000万円以上世帯は2万3,684円増と、低所得層ほど収入に対する影響が大きい。東京商工リサーチによれば円安倒産は26年上半期に45件と前年同期比1.3倍。帝国データバンク調査では2,290社の想定為替レートは平均147円87銭で、現状は14円もの想定差。「価格転嫁力の弱い卸売・小売中心に円安倒産は当面高水準」との観測が支配的で、政策側の対応（介入・利上げ・給付）の切迫度が増している。

参照元: フィスコ / みんかぶ

OPECプラス（有志国）が8月の生産目標引き上げに合意、供給拡大姿勢を鮮明にしたことで、米WTI原油先物は1バレル=70ドル割れの69ドル台まで下落（中東情勢の緊張後退も追い風）。日本市場では東証プライムの海運業・輸送用機器・石油・鉱業が値上がり率上位を占め、トヨタが+3.4%と3日続伸。一方、上流エネルギー株はやや軟調。円安と原油安の"良い組み合わせ"が輸出セクターに追い風となる構図。日本企業がイラン政府とイラン産原油の購入再開に向けた協議を開始したとの報道もあり、エネルギー地政学が再び動意含み。
