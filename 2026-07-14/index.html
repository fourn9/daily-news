本日最大Apple、iOS 27パブリックベータ公開 ─ AI搭載新Siriを一般ユーザーに開放、25億デバイス規模で"ChatGPT対抗"の実地テスト開始

参照元: TechCrunch (Sarah Perez) / Engadget (Will Shanklin) / The Verge / MacStories / 9to5Mac / CNET / Wired / BGR / Techmeme

Appleが7月14日、iOS 27・iPadOS 27・macOS 27 "Golden Gate"・watchOS 27・tvOS 27のパブリックベータを一斉公開、6月WWDCで発表したAI搭載新Siriを開発者以外にも初開放した。世界25億台のAppleデバイスの一部でも導入されれば"史上最大のAIアシスタント実地テスト"となる。新Siriはメール・写真・メッセージなど端末内情報にアクセスし、画面内容を認識、世界知識を根拠付ける（grounding）応答が可能で、ChatGPT/Gemini/Claudeへの本格対抗策と位置付け。Dynamic Islandからのスワイプ・Spotlight統合・専用アプリ化など全OSレベルで統合され、iPhone・iPad・Mac・Apple Watch・CarPlay・AirPods・Apple TV・Vision Proの全製品で利用可能。技術基盤はApple Silicon上で動くApple Foundation Models（Googleと共同開発、Geminiを蒸留したオンデバイスモデル）＋Private Cloud Compute。開発者ベータでは"ニュース検索"の指示に対しSiriが連絡先の"Iran"氏を検索する等の混乱もあったが、Photo Library検索・グループチャット要約・カレンダー自動追加・カメラ画像の栄養情報照会など基本タスクは大幅改善。秋の正式リリース前に消費者からのフィードバックを幅広く集める戦略。

Google DeepMind Hassabis CEO「フロンティアAI規制の独立機関を」 ─ FINRA型の"標準機関"提案、Anthropic Mythos／OpenAI Sol政府審査の恒久化、モデル公開30日前の任意事前レビュー

参照元: TechCrunch (Russell Brandom) / Demis Hassabis X投稿「A Framework for Frontier AI and the Dawning of a New Age」

Google DeepMinds CEO デミス・ハサビス氏が7月14日、X上に「A Framework for Frontier AI and the Dawning of a New Age」と題する長文を投稿、フロンティアAI規制の独立標準機関の創設を提案した。モデルは米金融業界自主規制機関FINRAで、フロンティアラボが公開30日前までにモデルを任意で提出→標準機関がテスト→評価プロトコルの有効性が確立次第、米市場への展開に必須化する段階的アプローチを提唱。現在は米政府がAnthropic Mythos・OpenAI Solを個別審査しているが、"技術的専門性の欠如"と"公開可否判断の不透明性"が批判されてきた経緯があり、それを恒久化する構想。政府が後ろ盾となりながらAIラボ資金で運営、オープンソース代表・技術専門家・AI安全性団体の外注評価も含む設計。ホワイトハウスAI顧問（a16zのSriram Krishnan氏）は「AI版FDAは行政内には作らない」と語っており、Hassabis案はその隙を突く自主規制型として提示された形。トランプ政権下でのAI規制議論の"新しい入口"として注目される。

NVIDIA×三菱重工、AIデータセンター提携 ─ "AIファクトリー"に日本の水冷・エネルギー管理技術を統合、電力・熱問題を日本製で解決

参照元: 日本経済新聞

米NVIDIAと三菱重工業が7月14日、AIデータセンター技術で提携すると発表。NVIDIAが世界各地で建設を進める次世代拠点"AIファクトリー"に、三菱重工の水冷システム・エネルギー管理技術の導入を検討する。両社技術を組み合わせ、AI向けデータセンターの電力消費と発熱を大幅に抑制する狙い。日本ではSKグループ（韓国）が主導する国内AIファクトリー計画も進行中で、日米韓連携の枠組みが形成されつつある。関連する日経記事によれば、NVIDIA GPU向け液冷需要拡大で富士電機は冷却電力85%削減の水冷機器を開発、三菱重工の既存水冷改良は電力最大7.6%削減を達成しており、日本の重電・化学プラント技術がAIインフラのボトルネック（消費電力・発熱）解消の切り札に浮上。1兆円規模のAIデータセンター市場で日本メーカーの存在感を高める契機となる可能性。

GMOインターネットG、在宅勤務を完全廃止 ─ 熊谷会長「タイピング数が減った、トータルでマイナス」、コロナ禍以降6年半継続の"週1推奨"も終了・出社原則に

参照元: 日本経済新聞 / ITmedia NEWS / GMOインターネットG 熊谷正寿氏X投稿

GMOインターネットグループは7月13日以降、6年半継続してきた在宅勤務制度を完全廃止した。熊谷正寿会長兼社長がXで説明、"在宅で生産性が上がる人がいる一方、時間当たりのタイピング数は減っており、トータルでは在宅勤務はマイナス"と判断根拠を明かした。GMOは2020年1月のコロナ禍を受けグループ全体で在宅勤務を導入、2023年に週2日の推奨を廃止、以降は"採用・QOL配慮"で週1回の在宅を認めていたが、13日にそれも撤廃、原則出社に統一。個別事情（家庭・オフィス環境）はグループ各社の裁量で対応。日本国内でも本田技研・楽天など出社回帰の動きが続いているが、"創業経営者が自らのXで完全廃止を宣言・タイピング数まで開示"という説明の生々しさが議論を呼び、NewsPicks上で224 Picksの反応。米AmazonやDell・Meta等の出社回帰と平仄が合い、コロナ禍で始まった"日本のリモートワーク大実験"が一区切りを迎えた象徴的発表となった。

7月14日の日本債券市場で金利が全域で急低下（価格急上昇）、20年債入札が"非常に好調"となったことが引き金となった。ある国内証券ディーリングルームでは午後0時半すぎ「これは事件だ」との驚きの声。前日は片山さつき財務相のGPIF等年金による国内資産投資支援発言→木原官房長官の"資産構成割合の変更を想定せず"否定→3市場トリプル安への逆流という展開だったが、14日は再びGPIF買い支え観測が優勢に。長期金利は日中2.755%→2.710%まで低下（前日は1996年以来の高値圏2.79%）、片山財相は"国債版NISA"導入への意欲も表明。厚生労働相・財務相のGPIF構成割合修正への同時言及も報じられ、いよいよ制度側の"クジラ動員"の準備が公式に動き始めた印象。仏興業銀行Albert Edwards氏の"日本版Truss moment"警告が現実に近づく中、"官製相場化"の色が強まる週初のリセット。米CPI下振れによる米金利低下も追い風に、AI・半導体売り／銀行株買いの日本市場の"新旧交代"ローテーションを一段と刺激。

ニューヨーク州、新規データセンター建設を全面凍結 ─ Hochul知事が緊急令、50MW以上の12件超が停止、AI電力需要への公的バックラッシュ本格化・全米初のモラトリアム

参照元: TechCrunch (Tim De Chant) / Kathy Hochul知事声明 / New York DEC / Pew Research / BloombergNEF

ニューヨーク州のKathy Hochul知事が7月14日、州として全米初の新規データセンター建設凍結の緊急令に署名した。対象は50MW以上のプロジェクト（該当12件以上）で、州環境保護局（DEC）は既に完了している許可以外は新規発行しない。凍結は環境レビュー完了まで（約1年見込み）継続、追加で"データセンター事業者から州電力網基金への拠出義務"や"ハイパースケール施設への税優遇除外"も検討中。Hochul知事は"進歩は電気代・水源枯渇・騒音公害と共に訪れるべきではない"と発言。背景はPew Researchの最新調査で"AI活用を歓迎する米国人はわずか10%"、"AI経済ブースト効果を信じる層は1/4未満"という世論のバックラッシュ拡大。既にニューヨーク州議会は20MW超で1年凍結・3年凍結案を審議中、Vermontサンダース議員は全国凍結を提案、Maine州は Mills知事が同種法案を拒否権発動していた経緯。BloombergNEFは2030年までに新設DCの約1/4が500MW超えと予測しており、"AI需要 vs 州レベル反発"の衝突が本格化。IBM/Meta等の"AI CapEx膨張"報道と対をなす、"AI経済の外部性が政治化した"象徴案件。

参照元: 日本経済新聞 / Reuters / NewsPicks / UAE政府 / トランプ大統領SNS投稿

トランプ米大統領が7月14日、前日SNSで表明したホルムズ海峡通過商船への20%通航料案を、わずか1日で撤回したと発表。「湾岸諸国との貿易・投資協定と置き換える」「投資は莫大なものになる」とSNSで説明、中東指導者らとの協議で決めたと明かしたが、詳細は不明。商船からの高額徴収に批判が集中したための後退と見られる。同日UAEはホルムズ海峡で自国タンカー2隻がイラン攻撃を受け、乗組員1名死亡・8名負傷と発表、非難声明を発表。米中央軍による対イラン攻撃は"1週間で4日連続"に達し、米国は海上封鎖の再開方針を明示。市場はこれを受け原油急伸、日本20年債入札の"クジラ買い"報道と併せて日経平均・韓国KOSPIは反発局面に転じるも、中東情勢は"タンカー・湾岸基地・海峡封鎖"の三重変数で不安定。オマーン仲介・カタール仲介（前日ハマド前首長死去で仲介役の不透明化）の実施可否とG7/OPEC+の対応が今週の焦点。トランプ氏の"通航料案→撤回"の即応朝令暮改は、政策の予見可能性リスクを改めて浮き彫りに。

参照元: 日本経済新聞 / ニチレイ / 日本KFCホールディングス

冷凍食品・食品物流大手ニチレイに7月14日までに不正アクセスがあり、影響で日本ケンタッキー・フライド・チキン（KFC）を始めとする外食チェーンの一部で食材調達が滞り、臨時休業や販売品目縮小が発生していると日経が報道。ニチレイは食材の一次卸・冷凍物流を担う基幹インフラで、影響範囲は外食のみならず小売・給食にも及ぶ可能性。ランサムウェア等の詳細は現時点で公表されていないが、①受発注・出荷管理システムの停止、②冷蔵冷凍倉庫の温度管理IoT機器への波及、③復旧に要する時間の不透明性──が焦点。前月のアサヒビール系サイバー攻撃、ニデック等の会計不正問題と併せ、日本の"生活インフラ企業"のサイバーレジリエンスが改めて問われる展開に。IBM CEOレターで指摘された"顧客企業のサイバーセキュリティ懸念によるIT予算配分の混乱"を裏付ける、日本での具体事例としても関心を集める。

滋賀県、ペロブスカイト太陽電池を体育館に全国初の本格商用導入 ─ 曲面・軽量の日本発技術がついに実用段階、政府の"次世代太陽電池"戦略の実証第1号

参照元: 日本経済新聞 / 滋賀県

滋賀県が県立体育館の屋根にペロブスカイト太陽電池を本格商用導入したと日経が7月14日報じた。全国の公共施設として初の本格商用ケースで、日本発の次世代太陽電池技術（軽量・曲面設置可・低コスト製造）の実用段階入りを象徴。日本政府は"シリコン太陽電池で中国に敗れた反省"を踏まえ、桐蔭横浜大の宮坂力特別栄誉教授が発明したペロブスカイトを国家戦略技術に位置付け、積水化学工業・パナソニック等が量産化に取り組んでいる。従来型太陽電池が困難だった体育館屋根等の軽量構造・曲面への導入が可能となり、都市部の建物垂直面・車体・農地の日陰活用など、"設置場所の常識を変える"応用が視野に。米中の関税・製造ボトルネックが続く中で、日本の"素材×製造技術"優位を活かした産業政策成功例となるかが試される。
